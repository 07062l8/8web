# LaHome Studio 

**La Home** design studio website designed to showcase portfolio, team and provide quote request form.
The project was completed for educational purposes and is posted on GitHub Pages:
 [La Home Studio](https://07062l8.github.io/8web/main.html)

## Project description
The site is a multi-page resource for a design studio.
It includes a home page, portfolio, studio and team information, a quote request form, and an FAQ section.
Implemented adaptive layout, support for light and dark themes, form validation and saving the latest application in **localStorage**.

## Project structure
- **main.html** — main page (services, benefits, contacts).
- **portfolio.html** — portfolio of projects.
- **about.html** - information about the company, founder and team.
- **estimate.html** - estimate request form with validation and the “Your last request” block.
- **index.html** — FAQ section.

**Files:**
- CSS and JS files are located in the project root.
- Media (images, sounds) are located in separate folders (`img/`, `sounds/`).

##  Main functionality
-  Switch light/dark theme.
-  Copy contacts to the clipboard.
-  Counters and scroll indicator.
-  Estimate request form: 
- Checking and validating fields. 
- Modal window for successful submission. 
- Saving the latest request in **localStorage** and displaying it.
-  Adaptive layout for mobile devices.

##  Technology
- HTML5, CSS3, JavaScript (jQuery).
- Bootstrap 5.
- GitHub Pages for publishing.
